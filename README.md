# Login-Register Project

## Overview
This is a **frontend-focused login and registration project** built with HTML, CSS, and JavaScript.  
It demonstrates **responsive UI, client-side validation, and basic authentication flow** using `localStorage` for demo purposes.

---

## Tech Stack
- **HTML** – Structure of the pages
- **CSS** – Polished and professional styling
- **JavaScript** – Client-side validation, form handling, password security
- **LocalStorage** – Simulated database for storing users

---

## Features
- **Responsive Login & Register pages**  
- **Password validation**: Minimum 6 characters, at least 1 number, 1 special character (`@ $ ! % * ? &`)  
- **Confirm Password check**  
- **Duplicate email detection** during registration  
- **Login validation**:  
  - Shows error if account not found  
  - Shows error if password is incorrect  
- **Inline error messages** for better user experience  
- **Clean & modern UI** (Figma/PSD style)
